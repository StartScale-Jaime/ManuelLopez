# 🏈 FootImproveness MVP

Una landing page profesional para conectar jugadores de fútbol con entrenamientos personalizados, desarrollada siguiendo los requerimientos del template optimizado.

## 🎯 Descripción del Proyecto

**FootImproveness** es una plataforma de contacto para jugadores de fútbol que buscan mejorar su rendimiento a través de entrenamientos personalizados, diseñados por un especialista graduado en CAFYD con máster en preparación deportiva.

## ✨ Características Principales

### 📄 Landing Page Atractiva
- Diseño moderno y responsive orientado a hombres jóvenes que juegan fútbol
- Hero section con call-to-action prominente
- Secciones informativas sobre los beneficios del entrenamiento personalizado
- Colores verdes y diseño deportivo que conecta con el público objetivo

### 📝 Formulario de Contacto Inteligente
- Modal con formulario estilo TypeForm
- Campos para información personal, nivel de juego, objetivos y preferencias de contacto
- Validación de formularios en tiempo real
- Mensaje de confirmación automático
- Experiencia de usuario optimizada para conversión

### 👨‍🎓 Página "Mi Historia Profesional"
- Información detallada sobre la formación académica (CAFYD + Máster)
- Especialización en preparación deportiva
- Filosofía de entrenamiento personalizada
- Diseño profesional que genera confianza

### 📞 Página de Contactos
- Múltiples métodos de contacto (email, teléfono, WhatsApp)
- Enlaces a redes sociales
- Horarios de atención claramente definidos
- Información de respuesta rápida

## 🚀 Tecnologías Utilizadas

- **Next.js 15** - Framework React para desarrollo full-stack
- **TypeScript** - Tipado estático
- **Tailwind CSS v4** - Estilos utility-first
- **Shadcn/UI** - Componentes de UI pre-construidos
- **Radix UI** - Componentes primitivos accesibles
- **Lucide React** - Iconos SVG optimizados

## 🛠️ Instalación y Desarrollo

### Prerrequisitos
- Node.js 18+ 
- npm o yarn

### Comandos de Desarrollo

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build

# Ejecutar en producción
npm start

# Linting
npm run lint
```

La aplicación estará disponible en `http://localhost:3000`

## 🎨 Diseño y UX

### Paleta de Colores
- **Verde principal**: `#16a34a` (green-600)
- **Verde claro**: Tonos de green-50 a green-100 para fondos
- **Grises**: Para texto y elementos secundarios
- **Blanco**: Para fondos de tarjetas y contrastes

### Tipografía
- **Fuente principal**: Geist Sans (moderna y legible)
- **Fuente mono**: Geist Mono (para código si fuera necesario)

### Responsive Design
- Mobile-first approach
- Breakpoints optimizados para todas las pantallas
- Navegación adaptativa
- Formularios optimizados para dispositivos móviles
